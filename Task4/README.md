# Why cost monitoring is important for beginners
Ans a) Cost Monitoring is Important for beginners because:
1) Beginners often experiment with AWS resources without realizing costs.
2) Monitoring ensures you do not overspend or go beyond Free Tier.
3) Alerts help prevent unexpected bills.

# What causes sudden increases in AWS bills
Ans b) Causes that increases AWS bills is that
1) Leaving EC2, RDS, NAT Gateway, or S2 storage running beyond Free Tier limits.
2) Using paid servies accidentlly like larger EC2 instances
3) Multiple test resources running simultaneously
